# 🏛️ Temple Adventure (C++)

## About

Temple Adventure Game is a simple text-based adventure game made in **C++**. The player explores a mysterious temple by making choices at different stages. Each choice leads to a different outcome, making the game fun and interactive.

## Features

* Simple text-based gameplay
* Multiple decision paths
* Different endings based on player choices
* Easy to understand code
* Beginner-friendly project

## Technologies Used

* C++
* Console Input/Output

## Logic Used

This project was intentionally kept simple for learning purposes. The entire game is built mainly using:

* `if`
* `else if`
* `else`
* `cout`
* `cin`

No advanced concepts like classes, functions, arrays, or pointers were used, making it easy for beginners to read, understand, and explain.

## How to Run

1. Open the project in any C++ IDE (CodeBlocks, VS Code, Dev C++, etc.).
2. Compile the source file.
3. Run the program.
4. Enter your choices and enjoy the adventure.

## Learning Outcome

This project helped in understanding:

* Conditional statements (`if`, `else if`, `else`)
* User input and output (`cin` and `cout`)
* Basic program flow
* Decision-making logic in C++
